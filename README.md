 OrangeHRM automation testing

tools 
  -Java 
  -selenium 4.38.0
  -cucumber 7.31.0
  -chromeDriver 
  -surefire pluggin to generate report

  
project structure
src/main/resources/features    feature files
src/test/java/org/example/stepDefs   step definitions
src/test/java/org/example/testRunner  test runner classes
pom.xml                        maven setup tools

how to run test script
-from code download zip file and extract manually
or 
-git clone https://github.com/ehab-mohammed/Orange-HRM-.git
-download maven dependencies
-open the test runner
-right click "run Runner"
